## 📖 About The Project

This portfolio is designed to be more than just an online resume; it is a conversion-focused platform demonstrating a unique blend of technical development and digital marketing skills. Built entirely without heavy frameworks, it utilizes modern, semantic HTML, CSS Grid/Flexbox, and Vanilla JavaScript to deliver a lightning-fast, premium user experience.

### Key Focus Areas:
- **Frontend Development:** Clean, structured code, and a sleek, modern UI.
- **Email Campaigning:** Highlighting automation, strategy, and design.
- **Copywriting:** Emphasizing persuasive, sales-driven content.

## ✨ Features

- **Dark Mode Aesthetic:** A premium dark UI featuring glassmorphism and vibrant gradient accents.
- **Fully Responsive:** Mobile-first approach ensuring a flawless layout on all screen sizes.
- **Interactive Filtering:** A dynamic portfolio grid with smooth category filtering (Frontend, Email, Copywriting).
- **CSS Animations:** Smooth scroll reveals, dynamic progress bars, and hover transitions.
- **Typewriter Effect:** An engaging Hero section to immediately hook visitors.
- **Contact Form Validation:** Frontend verification for the user-friendly contact form.
- **Custom CSS Grid Layouts:** Complex layouts handled gracefully without external grid libraries.

## 🛠️ Built With

- **HTML5:** Semantic architecture
- **CSS3:** Flexbox, CSS Grid, Custom Variables, CSS Animations
- **Vanilla JavaScript (ES6+):** DOM Manipulation, Intersection Observers, Event Listeners
- **Google Fonts:** [Inter](https://fonts.google.com/specimen/Inter) & [Outfit](https://fonts.google.com/specimen/Outfit)
- **FontAwesome:** Scalable vector icons

## 💻 Running the Project Locally

To get a local copy up and running, follow these simple steps.

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/portfolio-website.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd portfolio-website
   ```

3. **Open the index.html file**
   Simply open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, Edge) or use a local development server like VS Code's "Live Server" extension.
